# Full-Stack E-Commerce Platform

A production-grade e-commerce application with a complete feature set built with Next.js and Prisma.

## Features

- Cart management with optimistic UI updates
- Stripe payment integration with webhooks
- Real-time inventory updates via WebSockets
- Admin dashboard with sales analytics
- Redis-backed session management

## Architecture

Next.js App Router for SSR/ISR pages, Prisma ORM with PostgreSQL, Stripe for payments,
Redis for sessions and rate limiting, deployed to Vercel with edge middleware for auth.
